**Analyze a project:**
```
/meta-analyze
```

**Check automation status:**
```
/meta-status
```

**View metrics and ROI:**
```
/meta-metrics
```

**Rollback automation:**
```
/meta-rollback
```

## How It Works

1. **Intelligent Analysis** - AI agent reads your code and understands context
2. **Interactive Questions** - Asks about your pain points and preferences
3. **Cost Transparency** - Shows estimates before generating anything
4. **Custom Generation** - Creates tailored automation for your specific needs
5. **Metrics Tracking** - Tracks actual time saved to prove value
